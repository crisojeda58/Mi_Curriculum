# **App Name**: CodeCanvas CV

## Core Features:

- Bio Section: Display personal information, including name, contact details, and a brief bio.
- Project Portfolio: Showcase your projects with descriptions, links, and optionally, embedded demos.
- Education History: Detail your educational background, including institutions attended, degrees earned, and dates of attendance.
- Skills Showcase: List your technical skills (e.g., programming languages, frameworks, tools) with proficiency levels.
- Downloadable CV: Allow visitors to download your CV in PDF format.
- Contact Form: Implement a contact form for visitors to send you messages directly.
- Resume Tailor: AI powered tool, which takes any job posting URL, and recommends ways to rephrase your resume in order to align to that job's desired qualifications. 

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey professionalism and sophistication.
- Background color: Light lavender (#E6E6FA), creating a soft and clean backdrop.
- Accent color: Muted violet (#8A2BE2) for highlighting important information and interactive elements.
- Use clean and modern sans-serif fonts for headings and body text to ensure readability.
- Incorporate minimalistic icons to represent skills, technologies, and social media links.
- Implement a single-page layout with smooth scrolling for easy navigation.
- Use subtle animations on the website elements to highlight features as the user scrolls through your CV.
